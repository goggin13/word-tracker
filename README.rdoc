# Setup
download https://github.com/goggin13/infrastructure in adjacent folder.
`docker login`

# deploy
```
./docker_build_and_tag.sh
./docker_push_tag.sh # use `docker login` if auth fails
cd ../infrastructure
./login_web01.sh
cd word-tracker
./deploy.sh
```

# run locally
`./docker_start_app.sh`
`./docker_exec.sh # for rails testing`

`bundle exec rspec`
